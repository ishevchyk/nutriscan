# NutriScan — Backend

FastAPI + PostgreSQL backend. For architecture and endpoint docs, see `CLAUDE.md`.

## Prerequisites

- Python 3.11+
- PostgreSQL running locally (or reachable via `DATABASE_URL`)

## Setup

```bash
# from backend/
python3 -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt

cp .env.example .env
# then fill in .env — at minimum DATABASE_URL and SECRET_KEY

createdb nutriscan   # or create the database another way; nothing does this for you

alembic upgrade head
```

## Run

```bash
source .venv/bin/activate   # if not already active
uvicorn app.main:app --reload
```

The API is at `http://localhost:8000`. Check it's up:

```bash
curl http://localhost:8000/health
# {"status": "ok"}
```

Interactive docs (Swagger UI) are auto-generated at `http://localhost:8000/docs`.

## Migrations

After changing a model in `app/models/`, generate a new revision:

```bash
alembic revision -m "describe the change" --autogenerate
alembic upgrade head
```

Always review autogenerated migrations before running them — `--autogenerate` doesn't catch everything (e.g. renamed columns show up as drop + add).
